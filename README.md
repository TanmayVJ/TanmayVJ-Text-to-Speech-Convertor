## 📁 Files in this project

| File | Purpose |
|---|---|
| `Text_to_Speech_Combined.py` | Main application |
| `embed_images.py` | One-time tool to bake images into the app |
| `images_b64.py` | Auto-generated by embed_images.py (do not edit) |
| `mic logo.png` | Top-left logo image |
| `text icon.png` | Window icon |
| `speak.png` | Speak button icon |
| `download.png` | Save button icon |

---

## ▶️ Running the app normally (without building)

Make sure all image files are in the **same folder** as the script, then:

```
python Text_to_Speech_Combined.py
```

---

## 📦 Installing required libraries

Run this once in your terminal / command prompt:

```
pip install gtts playsound pyttsx3 pillow pyinstaller
```

> **Note:** On Windows, `playsound` version `1.2.2` works best.
> Install it with: `pip install playsound==1.2.2`

---

## 🖥️ How the app works

| Control | What it does |
|---|---|
| **VOICE** | Male or Female voice (English only via pyttsx3) |
| **SPEED** | Fast / Normal / Slow (English only via pyttsx3) |
| **ACCENT** | English / Hindi / French / Spanish |
| **Speak** | Plays the speech out loud |
| **Save** | Saves the speech as an `.mp3` file |

> When **ACCENT** is set to Hindi, French, or Spanish — the app uses
> Google TTS (gTTS) which requires an internet connection.
> Voice and Speed settings are ignored for non-English accents
> because gTTS does not support them.

## 🔨 Building a single .exe file (for Windows)

Follow these steps **in order**:

### Step 1 — Embed the images

Place all your `.png` image files in the same folder as the scripts, then run:

```
python embed_images.py
```

This creates `images_b64.py` which bakes all images into Python code.
You will see output like:

```
[OK] Embedded 'mic logo.png' as MIC_LOGO
[OK] Embedded 'text icon.png' as TEXT_ICON
[OK] Embedded 'speak.png' as SPEAK_IMG
[OK] Embedded 'download.png' as DOWNLOAD_IMG

Done! images_b64.py created.
```

### Step 2 — Build the .exe

Run this single command:

```
pyinstaller --onefile --windowed --icon="text icon.png" Text_to_Speech_Combined.py
```

- `--onefile` → everything packed into a single `.exe`
- `--windowed` → no black console window behind the app
- `--icon` → sets the taskbar/desktop icon

### Step 3 — Find your .exe

After building, look inside the `dist` folder:

```
dist/
  Text_to_Speech_Combined.exe   ← this is your final file
```

Share just this one `.exe` file — no images, no folders, nothing else needed.

## ❓ Troubleshooting

**App crashes immediately on launch?**
- Make sure you ran `embed_images.py` before building. Without `images_b64.py`, the exe cannot find the images.

**"No module named gtts" error?**
- Run `pip install gtts` and rebuild.

**Hindi/French/Spanish not working?**
- These accents need an internet connection (they use Google's servers). Check your internet.

**Voice sounds wrong on English?**
- pyttsx3 uses Windows built-in voices. Go to Windows Settings → Time & Language → Speech to install more voices.

**playsound error on Windows?**
- Run `pip install playsound==1.2.2` (newer versions have a known bug on Windows).
